# 伦理与数据原则

ChemWorld 的目标是研究 Agent 如何做实验决策，而不是生成现实世界实验方案。公开演示、数据集和
论文写作都应让读者清楚区分：哪些是虚拟环境观测，哪些是现实化学证据。

## 发布数据时

- 给 trajectory 标注任务、世界律、生成版本与成熟度。
- 不把 proxy/lite 模块输出包装成真实实验数据。
- 检查 Agent trace、解释文本和 metadata 是否泄露隐藏世界或个人信息。
- 不发布 private-eval salt、隐藏 seeds、oracle state 或可逆场景参数。
- 避免提供可直接执行的危险、违法或高风险现实流程。

## 接入外部系统时

ChemWorld 可以用于课程、算法比较和虚拟实验设计练习。若未来连接真实设备、机器人、实验室数据或
外部数据库，应重新进行权限、安全、隐私与领域审查；虚拟环境里的合法 Action 不能自动成为现实
设备指令。

更具体的模型与使用边界见[了解适用范围](limitations.md)。
