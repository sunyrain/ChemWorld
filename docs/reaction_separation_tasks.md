# 反应与分离流程

很多虚拟环境在反应结束时就给出分数；ChemWorld 的长流程任务要求 Agent 继续面对萃取、洗涤、
干燥、浓缩、结晶或蒸馏。目标是研究跨阶段决策，而不是只优化一个反应条件。

## 一条典型反应—纯化路线

```text
投料 → 反应 → 结束反应阶段
  → 加萃取剂 → 混合 → 静置 → 分相
  → 洗涤 / 干燥 / 浓缩
  → final assay
```

每一步都会改变物料、体积、能量、风险、成本或样品量。相分离和后处理依赖真实前序状态；例如没有
形成两相时，`separate_phase` 不会静默成功。

## 三类关键问题

| 问题 | 任务会观察什么 |
| --- | --- |
| 分配发现 | Agent 能否用有限测量学习隐藏相分配规律 |
| 纯度—回收率权衡 | 深度纯化带来的纯度收益是否值得物料损失和成本 |
| 长流程规划 | Agent 能否跨反应与下游阶段保持合法状态和目标一致性 |

高反应产率并不保证高最终得分：错误相选择、过度洗涤、物料 hold-up 或不必要测量都可能降低最终
回收率。可选任务见[选择一个任务](tasks.md)。
