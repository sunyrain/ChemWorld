# 适用范围与限制

ChemWorld-Bench 研究预算受限、部分可观测条件下的闭环实验决策。它不是现实产率预测器、分子模拟器、
商业流程模拟器、实验机器人控制软件或安全决策系统。

## 可以声明

- task、scenario、mechanism、observation、scoring 和 trajectory 使用版本化合同；
- Agent 只能通过公开操作、测量、成本、约束和历史记录与环境交互；
- 正式运行使用显式领域 provider，轨迹可以 digest 校验、确定性回放和指标重算；
- 经典、RL 和 LLM 方法可以进入同一实验与资源账本；
- 六任务的机理/构成律干预通过可执行性、可辨识性、非灾难性和守恒控制；
- 当前经典确认切片揭示了目标改善、风险控制与最小实质效应之间的冲突。

## 不可以声明

- 输出能够预测或指导真实反应、分离、谱图、装置设计或危险实验；
- 模型卡中的参考验证等于工业、法规或实验验证；
- 明确的运行路由等于全部物理模块达到专业模拟精度；
- Safe-GP 已通过完整四任务优越性规则；
- 单 seed SAC 开发结果代表 RL 性能或最佳 checkpoint；
- fake client、stub 或 replay 代表真实 LLM 性能；
- 公开世界结果证明跨机理、私有世界或现实实验泛化；
- 当前候选包是正式发表级 benchmark release。

## 模型边界

- 反应动力学是局部虚拟机制与速率律，不是数据库级反应预测；
- 物性、分配、设备和材料效应是版本化 benchmark 参数，真实名称不自动带来现实精度；
- HPLC、GC、UV–vis、IR、NMR、MS 和 final assay 是状态耦合的合成观测；
- 平衡、电化学、结晶、流动和蒸馏只在各模型卡的窄适用域内解释；
- cost/risk 是虚拟任务约束，不是采购报价、职业健康或法规指标；
- 整体任务成熟度按最弱必需模块聚合，当前注册任务仍为 `lite`；
- 电化学和平衡已有可执行构成律族，但主指标对自适应探索的稳定可辨识性仍未确认。

## 评价边界

评价把终点目标、任务主指标、在线 shaping、操作约束、过程成本、测量成本、有效性、交互能力和方法
资源分开报告。交互层级不进入单一总分：recipe-search 与 operation-level closed loop 只能作系统级
比较；若要归因于算法，必须处于相同交互层级并使用相同资源与信息合同。

Safe-GP 确认结果的连续流效应为 0.018752，低于预注册 SESOI 0.020000，所以联合规则失败。后续
五世界 β 诊断不能修复或替代确认结果。SAC 100k 运行只有一个任务和模型 seed，且 80k checkpoint
优于 100k。所有这些结果都必须按原证据层级报告。

## 机理与迁移边界

机理干预会改变实际执行的反应网络或领域构成律，并绑定 opaque hash 和精确回放上下文。这证明环境
能产生可审计的因果变化，但不证明 Agent 能识别机理、迁移策略或在现实实验中保持排名。真正的训练
环境主张需要 Train→未见 Bench/Bridge 的等预算对照，并允许结果为负。

## LLM 与隐私边界

真实 LLM 运行必须记录模型、请求参数、token、费用、重试、失败和谱图披露条件。assigned/masked
消融保持非谱图公开证据不变。系统不请求或保存私有逐字 chain-of-thought；结构化 evidence、hypothesis、
uncertainty 和 rationale 足以进行公开审计，但不能证明模型内部机理与文字解释一致。

## 安全与提交边界

本地学生 harness 是有界消息与进程边界，不是针对恶意代码的操作系统沙箱。运行不可信提交仍需外部
低权限、无网络、只读资源、CPU/内存/进程/墙钟受限的容器。任何真实实验决策都应由具有相应资质的
人员和机构安全流程独立审查。
