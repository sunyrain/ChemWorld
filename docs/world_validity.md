# 什么让一个虚拟化学世界具有意义

虚拟世界不需要假装覆盖全部现实化学，但必须足以产生可信的实验决策问题。ChemWorld 用四项原则
区分“有意义的因果世界”和只会返回分数的玩具函数。

## 1. 守恒

物料、相、能量、样品和设备状态不能凭空出现或消失。操作产生的 spent sorbent、condensate、vent、
hold-up 与取样损失应进入明确 ledger；失败事务要完整回滚。

## 2. 因果耦合

Action 通过声明的模型改变后续状态、观测、风险和成本。加热不是修改一个“温度特征”，而是推进
时间、能量与反应过程；测量也会耗样并影响之后可做的实验。

## 3. 部分观测

Agent 不能读取隐藏组成、速率常数或世界标签。它只能通过允许的仪器与公开历史获取证据；否则任务
会退化成直接读取答案。

## 4. 适用域声明

每个模型公开输入输出单位、有效范围、失败模式、来源与成熟度。`lite`、`reference_validated` 或
`professional_candidate` 描述证据范围，不被包装成通用现实精度。

## 还需要第五项：可反驳

世界、任务和结果都应能被回放、独立检查并允许失败。一个方法没有达到预注册实质阈值时，即使方向
看起来正确，也应保留为失败；一个 Provider 超出适用域时，也应显式报告而非静默 fallback。

与[结果可信链](release_integrity.md)。
